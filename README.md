# BrickBreaker Game

A Java implementation of the classic BrickBreaker arcade game built using object-oriented programming principles.

## Overview

This project recreates the classic BrickBreaker game where the player controls a paddle to bounce a ball and destroy bricks. The game demonstrates core Java programming concepts including object-oriented design, game loops, collision detection, and event-driven programming.

The project was developed as part of a Java coursework assignment focused on software design and interactive applications.

## Features

- Classic BrickBreaker gameplay
- Paddle movement and ball physics
- Brick collision detection
- Score tracking
- Multiple rows of bricks
- Game-over and win conditions
- Keyboard input handling
- Real-time rendering and updates

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Java GUI libraries
- Event-driven programming
- Collision detection algorithms

## Gameplay

- Use the keyboard to move the paddle
- Prevent the ball from falling below the screen
- Break all bricks to win the level
- Each brick collision increases the score
